# Website-Sederhana

Website ini adalah proyek pengembangan web dasar yang saya bangun untuk mempelajari struktur HTML, desain tampilan dengan CSS, dan integrasi backend menggunakan PHP dan MySQL.

## 🚀 Fitur Utama

*   **Sistem Autentikasi**: Fitur login pengguna untuk membatasi akses halaman.
*   **Desain Responsif**: Antarmuka yang dibuat agar terlihat rapi dengan CSS.
*   **Database Integration**: Pengelolaan data menggunakan MySQL dan koneksi PHP.

## 🛠️ Teknologi yang Digunakan

*   **Frontend**: HTML5, CSS3.
*   **Backend**: PHP.
*   **Database**: MySQL.

## 📂 Struktur Proyek

*   `login.html` & `login.php`: Halaman dan logika autentikasi.
*   `koneksi.php`: File penghubung ke database.
*   `style.css`: Pengaturan gaya visual website.
*   `tampilan_admin.php`: Halaman dashboard untuk admin.

## 💻 Cara Menjalankan Projek (Lokal)

1.  Pastikan XAMPP sudah aktif (Apache & MySQL).
2.  Pindahkan folder proyek ini ke dalam `htdocs` di folder instalasi XAMPP.
3.  Import file database `.sql` yang terkait ke phpMyAdmin.
4.  Buka browser dan akses melalui `localhost/Website-Sederhana`.
